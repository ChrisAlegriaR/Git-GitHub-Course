## üìñ Course, Branches and Project Description  
This repository is composed of three parts:

1. **Git & GitHub Course**:  
   In the **`course/`** folder, you will find my personal notes on the basic and practical concepts of Git and GitHub that I learned in the **SergieCode** course. This course provides the fundamental knowledge needed to understand and practice version control, including commit creation, branching, merging, rebasing, remote management, GitHub usage and collaboration workflows. These notes include step-by-step explanations, screenshots, and real command examples applied during the learning process.

2. **Branches**:  
   Instead of small coding exercises, this repository uses **multiple Git branches** created intentionally as part of the course. These branches were used to practice real Git workflows such as merging, resolving conflicts, switching between histories, performing rebases, rewriting commit history, and pushing to remote repositories. Each branch represents a different practical scenario used to reinforce version-control concepts.

3. **Documentation Assets**:  
   Throughout the course, I also generated different screenshots, Git logs, diagrams, and annotated examples that were used to document the steps performed during practice. These visual elements help illustrate how Git behaves internally and externally when performing each operation.
